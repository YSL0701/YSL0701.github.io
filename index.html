噗浪時光機

噗浪雖然有提供時光機功能讓人能夠指定時間看自己的帳號過去發的文章，但是要查看其他人過去的文章只能到對方的頁面滾動時間軸，如果對方的文章量很多或是想看的文章在很久以前，那只能不斷滾動直到找到為止，可能要花上半個小時甚至一個多小時也是有可能的，如果瀏覽器意外被關閉那就又要再找一次。

為了節省時間，所以我查看噗浪取得文章和回應的 API 和 request 參數，並且利用 Node.js 進行網路爬蟲來取得帳號的 user id，用 user id 和時間作為參數發出 requset 後取得文章資料，之後就可以用這些資料作為參數發出 requset 取得文章回應。

前端使用 Vue.js 和 SCSS 製作出 RWD 網頁，利用 AJAX 和伺服器端取得整理過的資料呈現到畫面上，並且用 localStorage 來做出儲存帳號和書籤的功能。

使用到的技術 : Vue.js Vue Cli Vue Router Vuex AJAX RWD SCSS Node.js Express web crawler Heroku Git

Plurk 留言搜尋

噗浪的好友數量多起來之後，未讀留言的數量常常會到上百個，如果只想看某個人在這些文章中的留言的話一則一則看實在太花時間，所以我使用 Node.js 進行網路爬蟲，利用 AJAX 非同步的特性併發 request 取得所有要查詢的帳號的 user id，再用這些 user id 併發 request 取得所有查詢帳號在指定時間內發的文章的 ID，接著用這些文章 ID 併發 request 取得所有文章的回應，過濾出只有目標帳號的回應還有該文章網址，整理之後傳給前端畫面呈現，使用時只要按一顆按鈕就能節省很多時間，這樣就不用花時間每篇都看了。

使用到的技術 : Vue.js AJAX Node.js Express web crawler Bootstrap Heroku Git

新北市 YouBike 查詢

政府資料開放平台的 API 可以取得新北市各個 YouBike 站點的最新使用情況，到 YouBike 站點前先輸入關鍵字查詢可以避免到了站點沒車可租，或是沒位可還的情形，查詢站點後還可以直接用 Google 地圖查看站點所在地。

雖然是開放平台的 API，但因為資料回傳的 header 沒有 Access-Control-Allow-Origin ，無法跨網域取得資料，所以用 Node.js 代理發送 request 再回傳給前端頁面才能取得資料。

使用到的技術 : Vue.js AJAX Node.js Express Bootstrap Heroku Git
